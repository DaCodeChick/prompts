This guide exists for **one job only**:

Create a **three-sheet character reference set** from the supplied character image(s).

The three sheets must be generated as **three separate standalone images**.

They must all use the **same supplied character reference image(s)** as the identity source.

They must **not** be merged together.

They must **not** continue onto one another.

They must **not** be presented as one package board, contact sheet, collage, or multi-sheet layout.

---

# MASTER HARD RULE — MANDATORY

Generate **exactly three separate images**.

These must be produced as **three separate sheets**:

1. **Core Reference Sheet**
2. **Face & Expressions Sheet**
3. **Clothing, Accessory, Colour & Material Sheet**

Each of the three sheets must be:

- a **brand new standalone image**
- generated from the **same supplied character reference image(s)**
- **not connected** to the other two sheets
- **not placed on the same page** as the other two sheets
- **not cropped out of another generated image**
- **not reconstructed from a combined board**

The three outputs must be treated as:

- **Generation 1 = Core Sheet only**
- **Generation 2 = Face & Expressions Sheet only**
- **Generation 3 = Clothing, Accessory, Colour & Material Sheet only**

If fewer than three images are generated, the result is wrong.

If more than one sheet type appears in the same image, the result is wrong.

If the three sheets are combined into one board, the result is wrong.

---

# SOURCE IMAGE RULE

Use the supplied character image(s) as the visual authority for:

- identity
- face
- age appearance
- hairstyle
- hair colour
- eye colour
- skin tone
- body proportions
- outfit construction
- accessories
- colour palette
- materials
- motifs
- visible species traits
- asymmetrical detail placement

If the supplied image is itself a busy reference sheet, treat it only as a **character-information source**.

Do **not** copy the source page layout.

Do **not** copy the source page structure.

Do **not** automatically reuse the source image's extra sections.

Ignore source-image sections that are irrelevant to the current sheet being generated.

---

# UNIVERSAL CHARACTER LOCK

Across all three sheets, preserve exactly:

- facial identity
- facial proportions
- apparent age
- hairstyle
- hair silhouette
- hair colour
- eye shape
- eye colour
- skin tone
- body proportions
- visible outfit construction
- accessory shapes
- jewellery
- footwear
- motifs
- species traits
- markings
- makeup
- asymmetrical details

Do not redesign the character.

Do not simplify the character.

Do not move ornaments or asymmetrical details to the wrong side.

Do not make the three sheets look like different versions of the same person.

All three sheets must clearly represent the exact same canonical character.

---

# UNIVERSAL PAGE STYLE

Unless the user explicitly asks otherwise, every sheet should use a consistent professional presentation style:

- 4:3 landscape
- approximately A4 landscape feeling
- pale warm light background
- restrained headings
- neat panel spacing
- subtle border or light ornamental frame
- polished anime production reference-sheet style
- clean linework
- controlled shading
- no environment background
- no photorealism
- no 3D-render look
- no chibi
- no comic-panel chaos

The three sheets may share a similar page-design language, but they must still remain **three separate images**.

# SHEET 1 — CORE REFERENCE SHEET

## Hard Rule

Generate **exactly one image** for this step.

That image must be a **4:3 landscape Core Reference Sheet**.

It must contain **only** these five main character panels:

1. **FRONT**
2. **3/4 VIEW**
3. **SIDE**
4. **BACK**
5. **PORTRAIT (NEUTRAL)** on the far right

## Required Title

`[CHARACTER NAME] — CORE REFERENCE SHEET`

## Required Layout

One 4:3 landscape page.

Panel order from left to right:

1. **FRONT**
2. **3/4 VIEW**
3. **SIDE**
4. **BACK**
5. **PORTRAIT (NEUTRAL)**

The rightmost portrait must be the largest panel.

The portrait must be a **large three-quarter-angle portrait**, not a strict side-facing head profile.

## View Rules

### FRONT
- true straight-on front view
- full body
- head to toe
- uncropped
- neutral standing pose

### 3/4 VIEW
- true full-body three-quarter view
- head to toe
- uncropped
- neutral standing pose

### SIDE
- true side profile full-body view
- not another three-quarter angle
- head to toe
- uncropped

### BACK
- true full-body back view
- not rear three-quarter
- head to toe
- uncropped

### PORTRAIT (NEUTRAL)
- large right-side portrait
- neutral expression
- same character
- clean framing
- three-quarter-angle portrait

## Do Not Include

Do **not** include:

- face detail panels
- eye detail panels
- lip detail panels
- expression grids
- colour palettes
- material guides
- accessory detail grids
- clothing detail panels
- footwear detail panels
- notes
- lore
- profile text boxes
- weapons breakdowns
- extra characters

If anything except the five required panels appears, the Core sheet is wrong.

---

# SHEET 2 — FACE & EXPRESSIONS SHEET

## Hard Rule

Generate **exactly one image** for this step.

This must be a **brand new standalone image**, not connected to the Core sheet.

It must be a **4:3 landscape Face & Expressions Sheet**.

## Required Title

`[CHARACTER NAME] — FACE & EXPRESSIONS SHEET`

## Required Layout

Create one 4:3 landscape page divided into:

1. **Left column** for detail close-ups
2. **Upper main area** for the six expression portraits
3. **Bottom row** for five face-angle portraits

## Left Column

### EYE DETAIL
- one large close-up of the canonical eye

### MAKEUP & LIP DETAIL
- one close-up of the canonical eye makeup
- one close-up of the canonical lips

## Expression Grid

Create a **2 × 3 grid** of six matching head-and-shoulders expression portraits.

Default expressions:

1. **Neutral**
2. **Cold Smile**
3. **Amused**
4. **Playful**
5. **Slightly Surprised**
6. **Soft Laugh**

All six must clearly look like the same character.

## Face Angles Row

Create five matching face-angle portraits labelled:

1. **Front**
2. **3/4 View**
3. **Side Profile**
4. **Looking Down**
5. **Looking Up**

The Side Profile must be a true side profile.

## Do Not Include

Do **not** include:

- full-body turnaround panels
- core front / side / back / 3/4 full-body views
- large portrait panels like the Core sheet
- colour palettes
- material guides
- accessory detail grids
- clothing detail panels
- footwear detail panels
- notes
- lore
- extra characters

If anything except the required face-and-expression sections appears, this sheet is wrong.

---

# SHEET 3 — CLOTHING, ACCESSORY, COLOUR & MATERIAL SHEET

## Hard Rule

Generate **exactly one image** for this step.

This must be a **brand new standalone image**, not connected to the Core sheet and not connected to the Face & Expressions sheet.

It must be a **4:3 landscape Clothing, Accessory, Colour & Material Sheet**.

## Required Titles

Top title:

`[CHARACTER NAME] — CLOTHING & ACCESSORY DETAIL SHEET`

Bottom title:

`[CHARACTER NAME] — COLOUR PALETTE & MATERIAL GUIDE`

## Required Layout

Create one 4:3 landscape page divided into two main sections:

1. **Upper section** = clothing and accessory detail panels
2. **Lower section** = colour palette, material guide, and notes

The upper section should take about the top two-thirds.

The lower section should sit underneath and take about the bottom one-third.

## Upper Section

Create a **2 × 4 grid** of **exactly eight** detail panels.

Default panel types:

1. **Hair ornament / defining head accessory**
2. **Main waist or torso emblem**
3. **Mantle / collar / shoulder material**
4. **Neck & chest detail**
5. **Cloak / coat / main fabric pattern**
6. **Back emblem & fastening detail**
7. **Sleeve / glove / garment edge**
8. **Footwear**

These may be adapted to suit the specific character design, but there must still be exactly eight detail panels.

## Lower Section

Create three columns underneath the upper grid:

### Left Column — COLOUR PALETTE
Include 8 to 10 labelled swatches and relevant small visual samples.

### Middle Column — MATERIAL & TEXTURE GUIDE
Show the character's actual important materials and textures.

### Right Column — NOTES
Include about five short visual-consistency notes.

## Do Not Include

Do **not** include:

- full-body turnaround panels
- core reference views
- large portrait panels
- eye detail panels
- lip detail panels
- makeup panels
- expression grids
- face-angle rows
- extra characters
- extra outfits
- alternate forms

If anything except the required clothing / colour / material sections appears, this sheet is wrong.

---

# UNIVERSAL QUALITY CONTROL

Before accepting the three-sheet set, check for the following across all three outputs:

- different-looking faces between sheets
- hair ornament switching sides
- asymmetrical details moving to the wrong side
- outfit changing between sheets
- colour palette not matching the actual design
- incorrect materials
- malformed hands or feet where visible
- malformed eyes
- broken anatomy
- extra panels appearing
- forbidden sheet sections bleeding into the wrong image
- two or more sheet types merged together
- fewer than three total images

If any of those happen, the result should be treated as failed and regenerated.
