STYLE-SPECIFIC VISUAL DIRECTIONS
1. Dragon Ball Z

Push toward a strong Akira Toriyama-inspired DBZ anime look.

Use:

bold, confident outlines
angular facial construction
strong jaw and cheek definition
sharper eyes
thick, graphic eyelids
harder cel shading
dramatic highlight shapes in hair
punchy, high-contrast rendering
classic shonen intensity
simplified but powerful facial forms

This should read immediately as DBZ, not generic action anime.

2. Bleach

Push toward a Tite Kubo / Bleach visual identity.

Use:

sleek, elegant linework
refined, elongated facial proportions
narrow, cool eyes
fashion-forward calm expression
clean, minimal but stylish shading
smooth hair flow with confident strand grouping
understated but sharp facial detail
chic, mature anime coolness

This should feel unmistakably Bleach.

3. Naruto

Push specifically toward a Masashi Kishimoto / Naruto look.

Use:

flatter, simpler cel shading
compact shonen facial construction
more grounded, practical eye rendering
smaller irises than glossy modern anime
thicker upper eyelids
chunkier hair clumps
cleaner, slightly rougher shonen linework
simpler mouth and nose treatment
early-2000s ninja manga / anime design logic
a recognizably Naruto-style facial build

Do not make this soft, glossy, or generic.
It must clearly read as Naruto.

4. My Hero Academia

Push toward a Kohei Horikoshi / My Hero Academia style.

Use:

energetic, expressive face construction
thicker and more varied line weight
bold eye shapes
stronger eyelash / lid definition
dynamic facial exaggeration
chunkier, more animated hair masses
graphic shadow placement
lively, heroic expression language
stylized but clean modern shonen rendering

It must clearly read as MHA, not just “modern anime.”

5. One Piece

Push toward an Eiichiro Oda / One Piece look.

Use:

much more stylized facial simplification
exaggerated eye construction
simplified nose and mouth
playful, highly readable expression shapes
more cartoonish proportion logic
elastic visual design language
clean bold lines
simplified but unmistakable Oda-style character appeal

This one should look the most caricatured and unmistakably One Piece.

6. Evangelion

Push toward a classic Evangelion / Gainax era anime look.

Use:

leaner, more restrained proportions
a slightly more serious, introspective expression
subtle, classic cel-anime structure
softer but controlled linework
less decorative rendering
understated shading
1990s anime facial logic
subdued, emotionally reserved presentation

This should read clearly as Evangelion, with that classic anime feel.

7. Violet Evergarden

Push toward a Kyoto Animation / Violet Evergarden rendering style.

Use:

refined, elegant facial anatomy
delicate linework
luminous, highly polished eyes
soft, graceful shading
more detailed hair rendering
gentle natural highlights
subtle realism in skin and fabric
beautiful, emotional softness
polished premium anime finish

This should feel luxurious, delicate, and unmistakably Violet Evergarden.

8. Magic Knight Rayearth

Push toward a CLAMP / Magic Knight Rayearth 1990s shoujo style.

Use:

very large sparkling eyes
ornate iris highlights
long elegant lashes
delicate nose and mouth
graceful, slender facial proportions
soft shoujo glamour
glossy, decorative hair rendering
classic 90s fantasy-shoujo beauty
refined romantic stylization

This one should read immediately as Magic Knight Rayearth / CLAMP.
