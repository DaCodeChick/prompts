ANATOMICAL FIDELITY SYSTEM

Every defining anatomical feature is part of the character's identity.

Treat these with the same importance as the face.

Never simplify, omit, redesign, shorten, duplicate, or replace them.

If any feature becomes partially hidden, maintain complete anatomical continuity.

Horns

Preserve exactly:

shape length curvature thickness ridges orientation spacing attachment points symmetry colour surface texture

Both horns must remain complete.

Never crop, merge or deform them.

Ears

Preserve exactly:

species length angle thickness position silhouette

Never replace fantasy ears with human ears.

Tail

The tail is a functional part of the skeleton, not an accessory.

attachment point length thickness profile taper curvature flexibility colour markings scales/furtip shape

Never:

ANATOMICAL PRIORITY

Anatomical correctness always takes precedence over dramatic posing.

If a composition, camera angle, gesture, or object interaction would create ambiguous anatomy, simplify the pose while preserving the emotional intent.

Prioritize:

• correct anatomy
• believable balance
• natural biomechanics
• clear silhouettes
• readable hands
• stable object interaction

over:

• exaggerated gestures
• extreme foreshortening
• unnecessarily complex poses
• excessive hand flourishes

The illustration should never sacrifice anatomical accuracy for spectacle.

Whenever possible, choose compositions that naturally display the body in a physically believable way.

• relaxed fingers
• stable support
• realistic manipulation
• believable weight transfer

Avoid:

• decorative finger poses
• exaggerated hand flourishes
• impossible grips
• tangled fingers
• excessive overlap
• awkward wrist angles

If a simpler hand position communicates the action more clearly, always prefer clarity over complexity.

Unless magic itself is the selected Passion, the hands should appear occupied with the craft rather than casting magical effects.

HAND COMPOSITION SYSTEM

Whenever the hands play a visible role in the illustration, the composition itself should prioritize anatomical readability before dramatic posing.

Choose hand poses that naturally expose the structure of the hand rather than hiding it.

Prefer:

• open palms
• relaxed fingers
• gentle natural curves
• fingertips resting lightly on surfaces
• stable object interaction
• clear separation between every finger
• wrists in comfortable neutral positions
• believable weight transfer

Avoid:

• buried hands
• fingers disappearing into clothing
• fingers hidden inside hair
• hands disappearing into fur
• wrapped fists around rounded objects
• heavy foreshortening
• overlapping fingers
• decorative flourishes
• ambiguous silhouettes
• impossible wrist rotations

If the selected Passion requires complex manual work, simplify the interaction rather than increasing visual complexity.

It is always preferable to show one perfectly readable hand than two visually confusing hands.

Whenever possible, every visible finger should remain individually identifiable.

The audience should immediately be able to count exactly five fingers on every visible human hand.

If a composition would naturally obscure the fingers, choose a different hand position that communicates the same action while preserving anatomical clarity.

OBJECT INTERACTION SYSTEM

Whenever the character physically interacts with another object, creature, person, tool or environment, the interaction must obey believable biomechanics.

Hands should appear to support, hold, push, pull, pet, carry, grip or manipulate objects exactly as a real person would.

Choose interactions that clearly display:

• finger placement
• thumb position
• pressure
• balance
• contact points
• weight distribution

Avoid interactions where important anatomy disappears inside:

• fur
• feathers
• hair
• cloth
• smoke
• magical effects
• liquids
• foliage
• crowds

If the object naturally obscures the fingers, adjust the pose instead of hiding the anatomy.

Clear anatomy always takes precedence over dramatic contact.
